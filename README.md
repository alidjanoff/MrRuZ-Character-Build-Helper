# MuRuZ Character Build Helper

An intelligent stat optimizer and formula-driven recommendation engine for MU Online builds.
This application helps players distribute their level-up and reset points optimally using real in-game formulas instead of guesswork.

---

## ✨ Features

* **Formula-Driven**: Uses actual game formulas for all 10 character classes
* **Multilingual Support**: English, Russian, Azerbaijani
* **Build Archetypes**: Farm, PvE (Boss), PvP recommendations
* **Bonus Point Detection**: Identifies inconsistencies in stat progression
* **Vitality Toggle**: Allows skipping default VIT allocation if gear compensates HP
* **Modern UI**: Clean, responsive, and readable interface

---

## 🚀 Getting Started

### Run Locally

1. Install Python
2. Install dependencies:

   ```bash
   pip install -r requirements.txt
   ```
3. Run the app:

   ```bash
   python main.py
   ```

---

## 📦 Build Executable (.exe)

1. Run:

   ```bash
   build_exe.bat
   ```
2. Find the output inside:

   ```
   dist/
   ```
